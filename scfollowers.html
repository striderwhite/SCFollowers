Help

You need to enter a Soundcloud API URL. To get this URL follow these instructions.
 Note: This requires you use Chrome.

Step 1 - Goto followers page

Navigate to your followers page (i.e https://soundcloud.com/strider_white/followers)

Step 2 - Dev Console

Open dev console (press F12) and follow these steps
  1. Press "Network" tab
  2. Press "XHR"
  3. Now within filter type "followers?offset"


Step 3 - Copy Url

Now refresh the page and copy the first item on the list.
  1. Right click -> copy -> copy link address


Step 4 - Modify url

Within the URL change paramater "&limit=12" to "&limit=200"

Thats it!
